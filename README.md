# gamesense.pub
![Forum Screenshot](screenshot.png)



## Requirements

* Node.js 18+
* npm

## Installation

Clone or download the project.

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Start production server:

```bash
npm start
```

Open in browser:

```text
http://localhost:3000
```
